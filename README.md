# Util-Extensions

A collection of useful browser extensions to enhance your browsing experience.

## Extensions

### [Float Note Extension](./float-note-extension/)

A translucent, floating note-taking editor that saves notes associated with the current page's URL. Resume your notes anytime you visit the same page.

**Features:**
- Floating editor with glassmorphism UI
- URL-based note storage
- Auto-save functionality
- Draggable interface
- Per-tab note resumption
