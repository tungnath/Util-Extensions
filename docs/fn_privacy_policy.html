📝 Privacy Policy

Float Note Extension

Last Updated: March 2026

Overview

Float Note ("the Extension") is committed to protecting your privacy. This Privacy Policy explains how the Extension handles your data.

Data Collection

The Extension does NOT collect, store, or transmit any personal data to external servers.

All data remains entirely on your device. The Extension only:

  • Stores notes you create locally in your browser
  • Reads the current page URL to associate notes with webpages
  • Operates entirely offline without any network requests

What Data is Stored

Local Storage

  • User-created notes - Text content you type into the editor
  • Page URLs - The webpage addresses where you create notes
  • Timestamps - When each note was created

Storage Location

All data is stored locally using Chrome's chrome.storage.local API, which means:

  • Data never leaves your device
  • Data is not synced to the cloud
  • Data is not shared with any third parties
  • Data persists only on your computer

Data Usage

The Extension uses stored data only for:

  1. Displaying your notes when you return to a webpage
  2. Allowing you to edit and manage your notes
  3. Providing the core functionality of the Extension

Data Sharing

Your data is never shared with anyone. The Extension:

  • Does not send data to external servers
  • Does not share data with third parties
  • Does not use analytics or tracking
  • Does not use advertising networks
  • Does not sell or monetize your data

Data Deletion

You have full control over your data:

  • Delete individual notes using the delete button in the Extension popup
  • Clear all notes for a webpage using the "Clear All" button
  • Uninstall the Extension to delete all stored notes
  • Clear browser data (note: this may also delete Extension storage)

Security

Your data is protected by:

  • Local storage on your device (no network transmission)
  • Browser's built-in storage security mechanisms
  • No third-party access or processing

Third-Party Services

The Extension does not use any third-party services, APIs, or libraries that collect data.

Changes to This Policy

We may update this Privacy Policy from time to time. Any changes will be reflected in the Extension's documentation.

Compliance

This Extension complies with:

  • Chrome Web Store policies
  • GDPR (General Data Protection Regulation)
  • CCPA (California Consumer Privacy Act)
  • Other applicable privacy laws
Summary: Float Note is a privacy-first extension. All your notes are stored locally on your device. We do not collect, track, or share any of your data.
